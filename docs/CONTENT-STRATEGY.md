## 🔄 CONTENT REPURPOSING

Ein Content-Stück → 7 Plattformen

```
                    ┌─────────────────┐
                    │  REDDIT POST    │
                    │  (Long-form)    │
                    └────────┬────────┘
                             │
         ┌───────────────────┼───────────────────┐
         │                   │                   │
         ▼                   ▼                   ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│   TWITTER       │ │   LINKEDIN      │ │   NEWSLETTER    │
│   Thread        │ │   Post          │ │   Deep Dive     │
└─────────────────┘ └─────────────────┘ └─────────────────┘
         │                   │                   │
         └───────────────────┼───────────────────┘
                             │
                             ▼
                    ┌─────────────────┐
                    │    TIKTOK       │
                    │    Video        │
                    └─────────────────┘
```

**Beispiel:**
- Reddit Post → Twitter Thread (kürzen)
- Reddit Post → LinkedIn Post (professionalisieren)
- Reddit Post → Newsletter (erweitern)
- Reddit Post → TikTok Script (visualisieren)

**Zeitersparnis:** 80% weniger Arbeit durch Repurposing.
